Bidirectional Pair Survival Report

This report joins the forward run (SMDG as query side, Txt-18 as corpus side) with the reverse run (Txt-18 as query side, SMDG as corpus side). It prioritizes file pairs whose signal remains high in both model prompt directions.

How To Read This

Overall survival

A rank-based composite. A pair scores high only if it remains strong when compared in both model directions. This is the close-reading queue.

Broad affinity

Uses the p95 score in both runs. High values suggest many strong cells in the matrix, not just one spectacular sentence match.

Local spike

Uses the single best sentence-pair score in both runs. High values point to possible quotation, formulaic phrasing, or a concentrated parallel.

Directional coverage

SMDG→Txt asks how well SMDG sentences find matches in Txt-18. Txt→SMDG asks the inverse. Asymmetry is expected because each side has different length and internal repetition.
